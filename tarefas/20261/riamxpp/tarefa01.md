## IDE
# Visual Studio Code
**Debugar**
  * **Breakpoints** - Ao clicar na margem esquerda, um ponto vermelho é criado. O código pausará a execução nesse ponto, permintindo analisar o estado da aplicação;
  * **Inspeção de variáveis** - Durante a pause, o painel lateral exibirá todas as variáveis locais e globais, além dos valores atuais.
  * **Painel de Debug** - Acessível pelo ícone da "joaninha" na barra lateral ou ```Crtl+Shift+D```
  * **Hover** - Ao passar o mouse sobre uma variável durante um breakpoint, VS Code mostra o valor dela naquele exato momento.

**Controle de versionamento**
  * **Stage/Unstage** - Você pode preparar arquivos (````git add```) clicando no ícone ```+``` ou preparar todos as mudanças de uma vez.
  * **Commit** - É possível escrever mensagens de commit diretamente na caixa de entrada e finalizar o commit com o botão "Commit".
  * **Remote(Push/Pull)** - A ferramenta facilita a sicronização com repositórios remotos, permitindo enviar (```push```) e baixar (```pull```) alterações com um clique.
  * **Branches** - Na barra de status, canto inferior esquerdo, é possível alternar entre branches, criar novas branches ou mesclá-los.
