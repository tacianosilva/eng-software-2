##  Projeto da Disciplina
Título: Sistema de Gestão de Assistência Técnica

Repositório: https://github.com/jadso/assistencia

##  Linguagem de Programação
A linguagem de programação utilizada no desenvolvimento do projeto será o Python, principalmente no backend da aplicação, em conjunto com o framework FastAPI. O Python foi escolhido por ser uma linguagem de alto nível, com sintaxe simples e de fácil compreensão, o que facilita tanto o desenvolvimento quanto a manutenção do sistema. Além disso, sua integração com ferramentas modernas, como o FastAPI, permite a construção de aplicações web rápidas, seguras e eficientes. No contexto deste projeto, o Python será responsável por implementar toda a lógica do sistema, realizar o processamento das requisições, manipular os dados e fazer a comunicação com o banco de dados SQLite.
A equipe possui conhecimento básico/intermediário, com experiência em desenvolvimento de aplicações simples, adquirido ao longo da graduação e por meio de atividades práticas, como projetos acadêmicos e exercícios de programação.

##  Framework
O framework escolhido para o desenvolvimento do projeto é o FastAPI, utilizado no backend da aplicação em conjunto com a linguagem Python. O FastAPI é um framework moderno e de alta performance, projetado para a criação de APIs de forma rápida e eficiente. Ele se destaca por utilizar recursos atuais do Python, como tipagem de dados, o que permite validação automática das informações e geração de documentação interativa de forma simples. No contexto deste projeto, o FastAPI será responsável por estruturar o backend, gerenciar as rotas da aplicação, processar as requisições dos usuários e realizar a comunicação com o banco de dados SQLite.
